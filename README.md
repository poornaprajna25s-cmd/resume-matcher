# 🚀 AI Resume Matcher

---

## 📌 Overview

AI Resume Matcher is an end-to-end Machine Learning web application that analyzes resumes and matches them with job descriptions using NLP techniques.

It helps users understand their job fit, identify missing skills, and explore suitable career roles.

---

## 🎯 Features

- 📄 Upload Resume (PDF)
- 🧠 Automatic Text Extraction
- 📊 Match Score Calculation
- ✅ Skill Extraction
- ❌ Missing Skill Detection
- 🎯 Job Recommendations
- 📊 Visual Dashboard
- 🎨 Clean UI Design

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- PDFPlumber  
- Pandas  

---

## 🧠 ML Concepts Used

- TF-IDF Vectorization  
- Cosine Similarity  
- Skill Matching Algorithm  
- Hybrid Scoring System  

---

## ⚙️ How It Works

1. Upload resume (PDF)  
2. Paste job description  
3. Text is converted into vectors  
4. Similarity score is calculated  
5. Skills are extracted and compared  
6. Missing skills are identified  
7. Job roles are recommended
